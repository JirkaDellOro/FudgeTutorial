FUDGE repository

Getting started

First of all, you need to set up the working environment. Here we will show you how to install the FUDGE repository on your environment and get the first glimpse of the FUDGE editor.

1. create a GitHub account


this is optional, as you will be able to download the repository locally even without an account. Note, however, that upcoming changes to the engine will not be applied and you will have to manually re-download new versions.

A new account can be createdhere.

2. download/clone FUDGE-Repository


Open the FUDGE repository via Code -> Clone or via GitHub Desktop and clone the repository. As mentioned earlier, you can simply download the repository as a ZIP locally to your working environment.

3. install FUDGE


In order to install FUDGE, you need to make sure that you have the following installed in addition to the local files:

  • download nodejs Nodejs - needed to install FUDGE and get it running.

  • TypeScript - This is optional, but recommended to write custom scripts in FUDGE. To install TypeScript globally, type 

    npm install -g typescript

    in the terminal. Alternatively, you can use JavaScript to write the scripts.

Once this is done, follow the steps below:

  • Extract the ZIP file into the desired folder.

  • Install the needed modules via 

    npm install --omit=dev
    in the terminal in the folder where FUDGE is included or just run the install.bat(Windows only). If the repository has been cloned, you can skip the first step.

4. Start editor


to open the FUDGE editor now, you have the following options:

  • type electron 

    editor/electron

    in the terminal

  • alternatively you can use 

    npm run-script run

  • use the FUDGE.bat file in the folder (Windows only)

User Interface

If the installation was completed successfully, you will now see an empty window with a browser console.
